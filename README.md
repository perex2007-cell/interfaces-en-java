# 📚 Sistema de Espacios Culturales - Medellín

Este proyecto implementa una interfaz `EspacioCultural` y varias clases que representan espacios culturales de Medellín, aplicando programación orientada a objetos en Java.

---

## 🚀 Historial de Commits

### 🟢 1. Crear interfaz base

```
feat: crear interfaz EspacioCultural con métodos base
```


### 🌿 2. Implementar Jardín Botánico

```
feat: implementar clase JardinBotanico
```

### 🏛 3. Implementar Museo de Antioquia

```
feat: implementar clase MuseoDeAntioquia
```

### 🔬 4. Implementar Parque Explora

```
feat: implementar clase ParqueExplora
```

---

## 🧱 Estructura del Proyecto

* `EspacioCultural.java` → Interfaz base
* `JardinBotanico.java` → Implementación de espacio cultural
* `MuseoDeAntioquia.java` → Implementación de espacio cultural
* `ParqueExplora.java` → Implementación de espacio cultural
* `Main.java` → Clase principal para ejecución

---

## 🧠 Conceptos aplicados

* Interfaces en Java
* Polimorfismo
